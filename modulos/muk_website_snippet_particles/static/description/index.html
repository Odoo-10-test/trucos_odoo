MuK Particle Snippet

Odoo Website Snippet

MuK IT GmbH - www.mukit.at

Overview

Adds a snippet to the Odoo Website. The snippet uses particlesJS to give a variety of customization options. It is also possible to add the particlesJS Animation as a page background. The snippet can be customized directly inside the editor. To customize the background just change the particle.json file to match your needs.
